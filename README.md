
# Security Monitoring & Incident Response System (Advanced)

## Features
- Log Analysis Engine
- Brute Force Detection Logic
- Incident Classification (High Severity)
- Alert Dashboard GUI
- Alert Clearing System

## How to Run
1. Install Python 3
2. Install Flask:
   pip install flask
3. Run:
   python app.py
4. Open browser:
   http://127.0.0.1:5000/
